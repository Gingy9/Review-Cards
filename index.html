Flashcard Review

What is HTML?
HTML stands for HyperText Markup Language and is used to structure web pages.
What is CSS?
CSS stands for Cascading Style Sheets and is used to style HTML elements.
What is JavaScript?
JavaScript is a programming language used to make web pages interactive.
What does `flexbox` do?
Flexbox is a CSS layout module that allows responsive alignment and spacing of elements.
What is a CSS transition?
A CSS transition smoothly changes CSS properties over a set duration.
What is the DOM?
The DOM (Document Object Model) represents HTML elements as objects in JavaScript.
What is a CSS grid?
CSS Grid is a layout system for creating complex two-dimensional layouts.
What is `hover` in CSS?
`:hover` is a CSS pseudo-class that applies styles when the mouse is over an element.
